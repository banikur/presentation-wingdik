# Logo Wing Pendidikan 600

Ganti file **`wingdik-logo.png`** di folder ini dengan logo resmi Anda.

- Format disarankan: **SVG** (atau tambahkan `wingdik-logo.png` dan ubah path di `app/page.tsx` komponen header).
- Nama file tetap **`wingdik-logo.png`** agar tidak perlu ubah kode.
- Ukuran tampilan di UI: ~48–56 px (header); SVG akan di-scale otomatis.

Setelah mengganti file, refresh browser (`Ctrl+F5` jika cache logo lama).
