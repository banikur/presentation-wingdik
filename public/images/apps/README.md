# Logo aplikasi terkait

Letakkan logo resmi (PNG/SVG) di folder ini dengan nama file yang sama dengan entri di `lib/pengadaan/app-logos.ts`:

| File | Aplikasi |
|------|----------|
| `sakti.jpg` | SAKTI |
| `sirup.webp` | SIRUP |
| `inaproc.svg` | INAPROC |
| `sikal.svg` | SIKAL |
| `logo-siman-landing-biru.png` | SIMAK BMN |
| `simtelog.svg` | SIMTELOG |

Saat ini berisi placeholder berlabel teks. Ganti file tanpa mengubah kode jika nama path tetap sama.
