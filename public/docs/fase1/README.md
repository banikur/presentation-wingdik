# Dokumen Fase 1

Letakkan file dokumen di folder subfase yang sama dengan gambar (`skadik`, `wingdik`, dll.). Aplikasi mendeteksi file secara otomatis.

## Format yang didukung

`.pdf`, `.doc`, `.docx`, `.xls`, `.xlsx`, `.ppt`, `.pptx`, `.txt`, `.csv`, `.zip`, `.rar`, `.7z`

## Contoh

```
public/docs/fase1/skadik/renbut-skadik-2024.pdf
public/docs/fase1/skadik/lampiran-tabel.xlsx
```

Gambar tetap di `public/img/fase1/{subfase}/`.

Setelah menambah file, refresh browser (Ctrl+F5).
